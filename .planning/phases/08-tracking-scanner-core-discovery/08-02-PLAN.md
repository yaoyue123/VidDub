    Create `backend/app/services/discovery_scanner.py` with the following design, following the exact patterns from `backend/app/services/channel_scanner.py`.

    **Key architectural decision per D-v5.1-01 (single coordinator loop):** Unlike ChannelScanner which creates one APScheduler job per channel, DiscoveryScanner creates a single coordinator job that ticks on a short interval (e.g. every 15 minutes) and checks which sources are due for scanning. This prevents thundering herd when many sources are configured.

    ```python
    """Phase 8: DiscoveryScanner -- Single-coordinator APScheduler scanner for all DiscoverySource types.

    Design (per D-v5.1-01, D-v5.1-02):
    - Single APScheduler coordinator job (not per-source jobs)
    - Coordinator ticks on short interval and scans sources that are due
    - Type dispatch: keyword -> yt-dlp search, creator -> yt-dlp channel scan
    - Per-source filters applied before dedup
    - Dedup against Video.youtube_id + DiscoveryResult.youtube_id
    - Stores new results as DiscoveryResult rows
    - Writes DiscoveryScanLog per scan
    """
    ```

    **DiscoveryScanner class:**
    ```python
    class DiscoveryScanner:
        """Single-coordinator APScheduler scanner for DiscoverySource items."""

        COORDINATOR_JOB_ID = "discovery_scanner_coordinator"
        DEFAULT_TICK_MINUTES = 15

        def __init__(
            self,
            max_concurrent: int = 3,
            tick_minutes: int = DEFAULT_TICK_MINUTES,
        ):
            self._max_concurrent = max_concurrent
            self._tick_minutes = tick_minutes
            self._scheduler: Optional[AsyncIOScheduler] = None
            self._semaphore = asyncio.Semaphore(max_concurrent)
    ```

    **Lifecycle methods** (patterned after ChannelScanner):

    `async def start(self) -> None`:
    - If already started, log warning and return
    - Create `AsyncIOScheduler()`
    - Register coordinator job with `IntervalTrigger(minutes=self._tick_minutes)`, `coalesce=True`, `max_instances=1`, `misfire_grace_time=None`
    - Start scheduler
    - Log "DiscoveryScanner started: coordinator tick every N minutes"

    `async def stop(self) -> None`:
    - Shutdown scheduler (`wait=False`)
    - Set `self._scheduler = None`
    - Log "DiscoveryScanner stopped"

    **Coordinator tick** (called by APScheduler):

    `async def _coordinator_tick(self) -> None`:
    - Load all enabled DiscoverySource items where `last_scanned_at` is null OR `last_scanned_at + scan_interval_hours` < now
    - Iterate each source, call `scan_once(source.id)` for each
    - Use `asyncio.gather` with the semaphore for concurrent scanning (respect `self._max_concurrent`)
    - Log summary: "Coordinator tick: scanned N sources"

    The query for due sources:
    ```python
    async with async_session_factory() as session:
        result = await session.execute(
            select(DiscoverySource).where(
                DiscoverySource.enabled.is_(True),
                # Not yet scanned OR past scan interval
                DiscoverySource.last_scanned_at.is_(None),
            )
        )
        never_scanned = result.scalars().all()

        # Due sources (beyond their scan interval)
        threshold = datetime.now(timezone.utc) - timedelta(hours=source.scan_interval_hours)
        result = await session.execute(
            select(DiscoverySource).where(
                DiscoverySource.enabled.is_(True),
                DiscoverySource.last_scanned_at.isnot(None),
                DiscoverySource.last_scanned_at < threshold,
            )
        )
        due = result.scalars().all()
    ```
    Combine `never_scanned + due` into the list to scan.

    **Core scan method:**

    `async def scan_once(self, source_id: int) -> ScanResult`:
    Uses semaphore, delegates to `_scan_once_impl`.

    `async def _scan_once_impl(self, source_id: int) -> ScanResult`:
    1. **Load source**: query `DiscoverySource` by id. Return `ScanResult(status="failed", error_msg="Source not found")` if missing or disabled.
    2. **Fetch via yt-dlp**: type dispatch:
       - `"keyword"`: use `get_ytdlp_wrapper().search(source.source_value, max_results=source.max_results_per_scan)`
       - `"creator"`: use `get_ytdlp_wrapper().get_channel_videos(source.source_value, max_results=source.max_results_per_scan)`
       - For unknown types, return `ScanResult(status="failed", error_msg=f"Unsupported source type: {source.type}")`
    3. **Set found_count** = len(fetched results)
    4. **Apply filters**: call `_apply_filters(fetched, source)` to filter by min_views, max_views, min_duration_sec, max_duration_sec, published_within_hours
    5. **Deduplicate**: call `_deduplicate(filtered)` to remove videos already in `videos` or `discovery_results` tables
    6. **Store results**: call `_store_results(deduped, source_id)` to insert as `DiscoveryResult` rows
    7. **Update last_scanned_at**: `update(DiscoverySource).where(...).values(last_scanned_at=now)`
    8. **Determine status**: if `error_msg` set, "failed"; if added_count < found_count and no error, "partial"; else "success"
    9. **Write scan log**: call `_write_scan_log(source_id, found_count, added_count, status, error_msg)`
    10. Log scan result and return `ScanResult`

    Add try/except around step 2 (yt-dlp fetch). On exception, write scan log with `status="failed"` and `error_msg`, return `ScanResult(status="failed", error_msg=str(e))`.

    **Filter method:**

    `def _apply_filters(self, videos: list[dict], source: DiscoverySource) -> list[dict]`:
    Mirror the ChannelScanner._apply_filters pattern but use DiscoverySource's filter columns:
    ```python
    out = []
    for v in videos:
        views = v.get("view_count")
        duration = v.get("duration")
        upload_date = v.get("upload_date")  # yt-dlp format: YYYYMMDD

        if source.filter_min_views is not None:
            if views is None or views < source.filter_min_views:
                continue
        if source.filter_max_views is not None:
            if views is not None and views > source.filter_max_views:
                continue
        if source.filter_min_duration_sec is not None:
            if duration is None or duration < source.filter_min_duration_sec:
                continue
        if source.filter_max_duration_sec is not None:
            if duration is not None and duration > source.filter_max_duration_sec:
                continue
        if source.filter_published_within_hours is not None:
            if not upload_date:
                # no date info, include by default
                pass
            else:
                try:
                    pub = datetime.strptime(upload_date, "%Y%m%d").replace(tzinfo=timezone.utc)
                    if (datetime.now(timezone.utc) - pub) > timedelta(hours=source.filter_published_within_hours):
                        continue
                except (ValueError, TypeError):
                    pass
        out.append(v)
    return out
    ```

    **Deduplication method:**

    `async def _deduplicate(self, videos: list[dict]) -> list[dict]`:
    - Collect all `youtube_id` values from videos (skip entries without youtube_id)
    - Query `Video` table: `select(Video.youtube_id).where(Video.youtube_id.in_(youtube_ids))` -- get set of existing video ids
    - Query `DiscoveryResult` table: `select(DiscoveryResult.youtube_id).where(DiscoveryResult.youtube_id.in_(youtube_ids))` -- get set of existing result ids
    - Return only videos whose youtube_id is NOT in either set
    - Log: "Dedup: N videos filtered (M existing)"

    **Store results method:**

    `async def _store_results(self, videos: list[dict], source_id: int) -> int`:
    - For each video, create `DiscoveryResult` row with:
      - `source_id = source_id`
      - `youtube_id = v.get("youtube_id", "")`
      - `title = v.get("title", "")`
      - `channel_name = v.get("channel", "")`
      - `duration_sec = v.get("duration")`
      - `view_count = v.get("view_count")`
      - `like_count = v.get("like_count")`
      - `thumbnail_url = v.get("thumbnail_url")`
      - `published_at = v.get("upload_date") or v.get("published_at")`
      - `discovered_at = datetime.now(timezone.utc)`
      - `status = "new"`
    - Use `session.add_all()` and commit
    - Handle `IntegrityError` per-entry (race condition on youtube_id uniqueness) with session rollback per entry, continuing to next
    - Return count of successfully added

    **Scan log method:**

    `async def _write_scan_log(self, source_id: int, found_count: int, added_count: int, status: str, error_msg: Optional[str]) -> None`:
    - Create and add `DiscoveryScanLog(source_id=..., found_count=..., added_count=..., status=..., error_msg=...)`
    - Commit, handle exceptions gracefully (log error, don't re-raise)

    **Global singleton accessor** (mirroring ChannelScanner pattern):

    ```python
    _scanner: Optional[DiscoveryScanner] = None

    def get_discovery_scanner() -> DiscoveryScanner:
        global _scanner
        if _scanner is None:
            _scanner = DiscoveryScanner()
        return _scanner

    def set_discovery_scanner(scanner: Optional[DiscoveryScanner]) -> None:
        global _scanner

    ```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-04 | Spoofing | yt-dlp extract results | accept | yt-dlp is a trusted dependency; malformed entries are skipped by ignoreerrors flag |

| T-08-07 | Information Disclosure | DiscoveryScanLog | accept | Single-user tool; scan logs contain only timestamps and counts, no video content |
| T-08-SC | Tampering | npm/pip installs | mitigate | No new dependencies beyond existing APScheduler, SQLAlchemy, yt-dlp |
</threat_model>
