## 小红书文案

**标题：** 为了更好的做自媒体，我开发了一个自动化搬运工具

---

正文：

我平时喜欢看一些优质的 YouTube 英文视频，但每次安利给朋友，对方一看是英文就划走了。

试过几个搬运工具，要么配音生硬得像念稿，要么折腾半天发现只能发 B 站，其他平台还得手动再传。有些还收费，一个视频收好几块钱。

后来就想，干脆自己写一个得了。

花了一段时间折腾，搞出来一个全自动的搬运管线，从下载到配音到发布全自动跑完，分享给有同样需求的朋友。

**它能做什么：**

把 YouTube 链接丢进去，自动把英文视频转成中文配音，然后一键发布到国内平台，全程不需要手动操作。

**整个流程是这样的：**

1. 自动下载 YouTube 视频（音频 + 画面一起拿）
2. 用 Whisper 把英文语音转成文字字幕
3. 调用 AI 把英文字幕翻译成中文
4. 用 CosyVoice2 合成中文配音，听起来比较自然
5. FFmpeg 自动对齐音轨，替换原声
6. AI 自动生成标题和标签
7. 自动发布到平台

**目前支持的平台：**
Bilibili、抖音、快手、腾讯视频、小红书，一共五个。登录一次之后会自动保持登录态。

**关于配音：**
用的是 CosyVoice2，不是那种机械的 TTS。语速、音调都可以调。如果你有比较多的素材，也可以训练自己的声音模型，然后用克隆声音来配音。

**使用体验：**
Web 界面操作，粘贴链接 → 选配置 → 点启动，然后就等着收工。可以在任务页面看实时进度，每一步走到哪了都清楚。配完不满意还能在字幕编辑器里手动调。

项目是完全开源的（MIT 协议），放在 GitHub 上，需要的话可以自己部署。对自媒体的朋友来说，省去了每天手动搬运的时间，精力可以放在内容本身。

有什么问题可以直接在 GitHub 提 issue，看到了会回。

🔗 项目地址：https://github.com/yaoyue123/VidDub
